# Historical Wonders Travel Website

---

# Project Overview

I recently completed a fully responsive, dark-themed travel website that highlights the Seven Wonders of the World, famous historical landmarks, and detailed travel guides.

This project was built entirely from scratch using HTML, CSS, and JavaScript, with a strong focus on immersive visuals, clean layout structure, and smooth user interactions.

---

# Key Features

* Glowing dark UI theme with golden-yellow highlights
* Smooth animations and engaging hover effects
* Interactive blog pages with search, filters, and pagination
* Modern gallery with fullscreen image viewer
* Dedicated Travel Guide pages
* Informative About page with structured content

* CSS
* JavaScript
